# 📦 pic-bed（图床仓库）

用于个人项目、博客和文档的图片存储仓库。

---

## 📁 目录结构

所有图片统一存放在 `images/` 目录下：

images/

📌 示例：
- images/avatar.png（头像）
- images/vue-intro.png（博客封面）
- images/login-ui.png（UI截图）

---

## 🚀 CDN 使用（推荐）

基础格式：

https://cdn.jsdelivr.net/gh/churjan/pic-bed/images/文件名.png

📌 示例：

https://cdn.jsdelivr.net/gh/churjan/pic-bed/images/avatar.png

Markdown 引用：

![图片](https://cdn.jsdelivr.net/gh/churjan/pic-bed/images/avatar.png)

---

## ⚡ GitHub 原始链接（备用）

https://raw.githubusercontent.com/churjan/pic-bed/main/images/文件名.png

📌 示例：

https://raw.githubusercontent.com/churjan/pic-bed/main/images/vue-intro.png

Markdown 引用：

![图片](https://raw.githubusercontent.com/churjan/pic-bed/main/images/vue-intro.png)

---

## 📌 文件命名规范

### ✅ 推荐
- user-avatar.png
- blog-vue-intro.png
- login-ui-v1.png

📌 示例含义：
- user-avatar.png → 用户头像
- blog-vue-intro.png → 博客封面
- login-ui-v1.png → 登录页面 UI

### ❌ 不推荐
- 图片1.png
- 最终版.png
- aaaa.png
- 中文或空格

---

## 🛠️ 使用方法

1. 上传图片到 `images/` 目录  
2. 使用规范命名文件  
3. 复制 CDN 链接  
4. 在项目 / 博客中引用  

---

## 📌 使用场景

- 博客文章配图  
- 前端项目资源  
- README 文档截图  
- UI 设计展示  
- 个人项目资源管理  

---

## 🧾 项目用途

统一管理图片资源，方便在开发与内容创作中快速引用。
